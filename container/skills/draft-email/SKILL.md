# Draft Email

Draft emails in Shintaro's voice. Does NOT send — only drafts for review.

## When to Use

- "Draft an email to..."
- "Write an email about..."
- "Help me email [person]"
- "Compose a message to..."

## Process

### 1. Gather Context

If not provided, ask:
- **Who** is the recipient? (name, relationship)
- **What** is the purpose? (follow-up, request, intro, thank you)
- **Tone?** (casual, warm professional, formal)
- **Key points** to include?

### 2. Load Voice

Reference the **persona** skill for Shintaro's full communication style. Key rules:

**Openings:**
- Peers/friends: "Hey [Name] --" (double dash, not comma)
- More formal: "Hi [Name],"
- Common openers: "Hope all is well", "Hope you're having a great weekend"

**Closings:**
- Casual: "Cheers, Shintaro"
- Professional: "Best, Shintaro"
- After a request: "Thanks, Shintaro"

**Signature phrases (use naturally, don't force):**
- "Loved hearing about..."
- "Important to me that..."
- "Would be fun to..."
- "candidly"
- "for what it's worth"
- "happy to discuss... if that would be helpful"

### 3. Email Format Rules

- Short paragraphs, lots of white space
- 1-2 exclamation points max (genuine enthusiasm only)
- NO em dashes ever
- No corporate jargon, no passive voice, no hedging
- Never use: "Per our conversation", "Circle back", "Synergies"
- One clear ask per email
- Front-load the important info

**Structure:**
1. Warm opener acknowledging context
2. Get to the point quickly
3. Clear action items or next steps
4. Generous closing (offer help, open door)

### 4. Determine Email Type

| Type | Approach |
|------|----------|
| Cold outreach | Short, specific ask, easy to say yes |
| Follow-up | Reference prior conversation, clear next step |
| Introduction | Warm, brief context on both parties |
| Thank you | Genuine, specific, forward-looking |
| Request | Clear ask, context for why, easy action |

### 5. Generate and Present

Output the draft in a code block with subject line:

```
Subject: [Clear, specific — under 50 characters]

Hey [Name] --

[Body]

Cheers,
Shintaro
```

Ask: "Want me to adjust the tone, length, or any specific part?"

## Anti-Patterns

- Don't send emails, only draft
- Don't write walls of text
- Don't use generic openings ("I hope this email finds you well")
- Don't bury the ask at the end
- Don't use AI-sounding language (run through de-ai patterns mentally)
- Don't copy signature phrases verbatim from samples without adapting to context
